黑色一眸
===========

黑传说的私人日志，部署在隔壁。jobinson99.ga


## 所有的设置 ##

所有的网站设置相关，也在此设置，因为这里足够复杂，几乎所有的类型都有，而且因为是私人的事，不想公开，搜索引擎优化啥的不重要，可以任意折腾。

下面列出和jekyll默认不同的部分

### 个性页面部分： ###

+ 首页：index.html
+ 友情链接：friends.html
+ 网络资源：resource.md
+ 域名指向：CNAME
+ 配置：_config.yml
+ 页面不在：404.html
+ 小图标：favicon.ico
+ 按时间列表：archive.html 时间轴

### 个性文件夹： ###

+ static
+ _posts
+ _sites
+ _includes
